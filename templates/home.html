AI Assistance for Humanity

Decentralized Intelligence Assistance

Most businesses (e.g. Facebook, Google, Microsoft, the banking industry) track, persist, and use our behavior for their profit.

Artificial Intelligence (AI) is dramatically enhancing our experiences resulting in profits windfalls to investors at the cost of many low skilled jobs (~80% of the job market).

Blockchain based applications and cryptocurrencies like Bitcoin are helping to wrestle some control from these centralized organizations although they are largely used on servers and distributed ledgers are still logically centralized.

Smart phones, our primary means of global communication and collaboration, are weak in maintaining our anonymity and privacy - critical to ensuring individual freedom.

What's needed are decentralized applications that run purely on mobile phones without using servers of any kind and share only what the owner specifically allows.

Synaptic Celerity™ is building a secure open-source decentralized AI peer-to-peer mobile application platform with end-to-end encryption for creating easy to use secure decentralized peer-to-peer smart mobile apps requiring no server connections that can be used around the world by any person for personalized AI while looking to protect their communications and personal data from unjust intrusion, interception and monitoring yet isolate bad actors from using the network.

Platform

Mobile Application Platform

Building a real-time decentralized personal AI mobile application platform is quite tricky. Today mobile applications largely talk with servers (e.g. Facebook, Instagram, Twitter) to store and retrieve data. Persistence and processing can be used on mobile phones but phones do not have the capacity of servers therefore they need to share resources while not having the addressability as servers.

This is a complex endeavor but needed to reduce dependency on centralized parties, provide transparency where needed, yet ensure personal privacy. To begin something complex, we must build a platform from which future functionality can depend.

Synaptic Celerity's platform provides mobile application infrastructure to build on but we do it a little differently than others such as Ethereum. We use our Foundation's Development team to develop the code with partnered entities to ensure high quality implementations vs. letting others post code to the platform.

The following features are provided or currently in development:

Conscious

Directed Acyclic Graphs (DAG) + Rete + Neuro-evolution - a form of machine learning that uses evolutionary algorithms to train artificial neural networks. This is the nervous system of the platform and its primary service. It's expected to be driven long-term towards an AGI and ASI.

Contracts

Think Ethereum turing-complete platform on a mobile using OpenDocuments for the contract language and a Rete rules engine for execution. This supports using the DIA service to truly make contracts 'smart'.

Info Vault

The platform allows for the user to grant access control to their personal information on their mobile if desired. Access can be revoked when its no longer desired for them to have it. Applications no longer need to know everything about you. This will end widespread data breaches.

Token Wallet

Maintaining tokens secure on your mobile.

Life IDentifier™

Pluggable, decentralized, people-controlled universal ID, called LID or Life IDentifier, that is unique to every life on the planet. Its primary components are:
Crypto Authentication (CA)
Reputation Based Access Control (RepBAC)
Circles of Influence

Recall™

Provides the working-, short-, and long-term memory for the DIA and other services requiring access to memory using caching, graphs, and a highly available/highly partition tolerant/tunable consistency persistence.

Amino™ - Utility Token

Aminos are unlimited tokens used in the platform as a method to manage resources (network, cpu, storage) to support it.

Syn™ - Foundation Token

Limited tokens used in the platform as a method to determine transaction fee distribution.

UBI

Universal Basic Income - Amino tokens for participating in the system to prevent double spend.

Sensors

Mobile-to-Mobile (M2M), End-to-End Encrypted, Double Ratchet Key Management, Mesh Network Capable, Anonymous Network, with Bad Actor Isolation.

Content

Full-stack synched decentralized javascript web applications on android for decentralized content distribution including static resources.

Consensus

Permissioned (LID) blockchain on Android. Prevent double spend. Anyone can get on but are severely restricted in what they can do until they create their LID. As their reputation grows, their access will too. Their reputation falls, so does their access.
